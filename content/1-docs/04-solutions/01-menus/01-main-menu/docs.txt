Title: Main menu

----

Text:

This is a very basic menu.

```php
<?php

// main menu items
$items = $pages->visible();

// only show the menu if items are available
if($items->count() > 0):

?>
<nav>
  <ul>
    <?php foreach($items as $item): ?>
    <li><a<?php ecco($item->isOpen(), ' class="active"') ?> href="<?php echo $item->url() ?>"><?php echo html($item->title()) ?></a></li>
    <?php endforeach ?>
  </ul>
</nav>
<?php endif ?>
```
