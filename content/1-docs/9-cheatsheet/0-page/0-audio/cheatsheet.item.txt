Title:

$page->audio()

----

Excerpt:

Returns a $files collection with all audio files in the page's directory

----

Return:

type: $files
text:

----

Text:

## Example

```php
<h2>Songs</h2>
<?php foreach($page->audio() as $audio): ?>
<audio controls>
  <source src="<?php echo $audio->url() ?>" type="<?php echo $audio->mime() ?>">
</audio>
<?php endforeach ?>
```
