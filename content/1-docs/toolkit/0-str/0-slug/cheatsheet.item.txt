Title: str::slug

----

Excerpt: Convert a string to a safe version to be used in a URL

----

Call: str::slug($string, $separator = '-')

----

Return: type: string
text: The safe string

----

Params: - 
  name: $text
  type: string
  text: The unsafe string
- 
  name: $separator
  type: string
  text: >
    To be used instead of space and other
    non-word characters.
